================================================================================
  IMPORTANT: OpenAI API Key Required
================================================================================

To use SpeakEase, you MUST configure your OpenAI API key.

📌 STEPS TO GET STARTED:

1. Get your OpenAI API Key:
   → Visit: https://platform.openai.com/api-keys
   → Sign up or log in to your OpenAI account
   → Click "Create new secret key"
   → Copy the key (starts with sk-)

2. Add the key to your .env file:
   → Open the .env file in the project root
   → Find the line: VITE_OPENAI_API_KEY=your-openai-api-key-here
   → Replace "your-openai-api-key-here" with your actual key
   → Save the file

   Example:
   VITE_OPENAI_API_KEY=sk-abc123xyz456...

3. Run the app:
   npm run dev

================================================================================

⚠️  WITHOUT THE API KEY, THESE FEATURES WILL NOT WORK:
   - Speech-to-text transcription (Whisper API)
   - AI feedback and scoring (GPT-4o-mini)
   - AI Tutor chat functionality
