# SpeakEase - Quick Start Guide

## 1. Install
```bash
npm install
```

## 2. Configure OpenAI API Key

Edit `.env` file:
```env
VITE_OPENAI_API_KEY=sk-your-actual-api-key-here
```

Get your key: https://platform.openai.com/api-keys

## 3. Run
```bash
npm run dev
```

Open: http://localhost:3000

## 4. First Steps

1. **Sign in** with Google
2. **Complete** a beginner lesson
3. **Record** your speech
4. **Get** AI feedback
5. **Track** your progress

## Key Features

### Speech Practice
Click lesson → Read text → Record → Get feedback

### Daily Challenge
Dashboard → Click challenge card → Record → Earn XP

### AI Tutor
Chat tab → Start session → Type message → Practice conversation

### Leaderboard
Leaderboard tab → See rankings → Track position

### Analytics
Analytics tab → View progress → Check scores

## Troubleshooting

**Microphone not working?**
- Grant browser permissions
- Check system settings

**API errors?**
- Verify OpenAI key in `.env`
- Check API credits

**Login issues?**
- Check internet connection
- Try incognito mode
